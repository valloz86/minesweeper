Для локальной сборки и запуска нужно 

1) Выкачать проект и открыть его в Идее (рекомуендуется использовать Java 17)
2) Нажать в Maven install и дождаться его выполнения
3) Запустить main в классе minesweeper.service.Application и дождаться Started Application в логах

После этого можно подставлять урл локалхоста c добавлением /api (дефолтный - http://localhost:8080/api) во фронтенд и начинать игру.
